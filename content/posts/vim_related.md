## Vim 相关技巧

> 收集一些常用的 Vim 相关技巧，包括插件、快捷键、命令等。(using lazyvim)

### Nice Motion
- `f{char}`: 移动到下一个 `{char}` 出现的位置, 类似于 `t{char}`
- `F{char}`: 移动到上一个 `{char}` 出现的位置, 类似于 `T{char}`
- `;` 和 `,`: 重复上一次的 f, F, t, T 命令

### Plugins

- lualine: vscode like status line
- gitblame: show git blame history
