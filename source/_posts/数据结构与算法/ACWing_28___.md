## 需要思考的就只有第三题


有 **n**n 个小朋友，编号 **1****∼****n**1∼n。

每个小朋友都拿着一个号码牌，初始时，每个小朋友拿的号码牌上的号码都等于其编号。

每个小朋友都有一个幸运数字，第 **i**i 个小朋友的幸运数字为 **d****i**di。

对于第 **i**i 个小朋友，他可以向第 **j**j 个小朋友发起交换号码牌的请求，当且仅当 **|****i****−****j****|****=****d****i**|i−j|=di 成立。

注意，请求一旦发出，对方无法拒绝，只能立刻进行交换。

每个小朋友都可以在任意时刻发起任意多次交换请求。

给定一个 **1****∼****n**1∼n 的排列 **a****1****,****a****2****,****…****,****a****n**a1,a2,…,an。

请问，通过小朋友相互之间交换号码牌，能否使得第 **i**i 个小朋友拿的号码牌上的号码恰好为 **a****i**ai，对 **i****∈****[****1****,****n****]**i∈[1,n] 均成立。

#### 输入格式

第一行包含整数 **n**n。

第二行包含 **n**n 个整数 **a****1****,****a****2****,****…****,****a****n**a1,a2,…,an。

第三行包含 **n**n 个整数 **d****1****,****d****2****,****…****,****d****n**d1,d2,…,dn。

#### 输出格式

共一行，如果能做到，则输出 `YES`，否则输出 `NO`。

#### 数据范围

前 **6**6 个测试点满足 **1****≤****n****≤****10**1≤n≤10。
所有测试点满足 **1****≤****n****≤****100**1≤n≤100，**1****≤****d****i****≤****n**1≤di≤n，保证 **a****1****∼****a****n**a1∼an 是一个 **1****∼****n**1∼n 的排列。

#### 输入样例1：

```
5
5 4 3 2 1
1 1 1 1 1
```

#### 输出样例1：

```
YES
```

#### 输入样例2：

```
7
4 3 5 1 2 7 6
4 6 6 1 6 6 1
```

#### 输出样例2：

```
NO
```

#### 输入样例3：

```
7
4 2 5 1 3 7 6
4 6 6 1 6 6 1
```

#### 输出样例3：

```
YES
```


```c++
#include <iostream>
#include <algorithm>
#include <vector>

using namespace std;
const int N = 110;
int p[N],n,a[N],d[N];
vector<int> A[N],B[N];

int find(int x){
  if(x!=p[x]) p[x]=find(p[x]);
  return p[x];
}

void merge(int a,int b){
  if(a<0||a>=n) return;
  p[find(a)]=find(b);
}

int main(){
  cin>>n;
  for(int i=0;i<n;i++) p[i]=i;
  for(int i=0;i<n;i++){
    int k;
    cin>>k;
    a[i]=k-1;
  } 
  for(int i=0;i<n;i++) cin>>d[i];
  
  for(int i=0;i<n;i++){
    merge(i-d[i],i);
    merge(i+d[i],i);
  }
  
  for(int i=0;i<n;i++){
    // 根据给得d[I] 可以得到得并查集
    A[find(i)].push_back(i);
    // 根据给得d[I] 得到的目标的并查集
    B[find(i)].push_back(a[i]);
  }
  
  // 判断两边的并查集是否相等 如果相等则 可以通过任意次变换得到目标结果
  for(int i=0;i<n;i++){
    sort(B[i].begin(),B[i].end());
      if(A[i]!=B[i]){
        puts("NO");
        return 0;
      }
  
  }
  puts("YES");
  return 0;
}
```
