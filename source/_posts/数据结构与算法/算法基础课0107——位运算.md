# 位运算

## 位运算的基础操作


* 题目1: 数二进制位中1的个数

```
#include <iostream>
#include <algorithm>

using namespace std;

const int N = 1e5+10;

int a[N];

int main(){
  int n;
  cin>>n;
  for(int i=0; i<n;i++){
    int x;
    scanf("%d",&x);
    int ans=0;
    while(x){
      x&=x-1;
      ans++;
    }
    printf("%d ",ans);
  }
  puts("");
  return 0;
}
```
